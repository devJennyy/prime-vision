# PrimeVision

A movie streaming and download platform featuring a sleek design, advanced search options, and a seamless user experience. Built with React, TypeScript, and Tailwind CSS, with data sourced from the YTS API. Hi-fi prototypes for both the website and mobile app were designed using Figma.

> [!IMPORTANT] 
> **This platform is not intended for any commercial purposes. It is solely for showcasing my Frontend Development skills.**

## Commands

1. Install dependencies:

```bash
npm i
```

2. Start the development server

```bash
npm run dev
```
