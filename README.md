# PrimeVision

A movie streaming featuring a sleek design, advanced search options, and a seamless user experience. Built with React, TypeScript, and Tailwind CSS. Hi-fi prototypes for both the website and mobile app were designed using Figma.

## Commands

1. Install dependencies:

```bash
npm i
```

2. Start the development server:

```bash
npm run dev
```
